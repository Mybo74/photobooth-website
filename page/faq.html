Questions fréquentes

Comment fonctionne le service de photobooth?

Prenez contact avec nous et donnez nous des details sur votre évenement, tels que la date, le type 'evenement ,
le nombre de convives, nous nous ferons une proposition repondant a vos v=beosin

Quels types d'évenements sont adaptés pour le Photobooth?

Tous types d'évenements: mariage, bapteme, anniversaire, baby shower, bar mitsva, evenement d'entreprise, actions commerciale,...

Quelles options de personnalisation sont disponibles pour les photos?

Differentes options de personnalisations : les frames sur les photos imptimés, l'écran d'accueil du photobooth,
les accessoires pour la pose photo

Quels espace est nécessaire pour l'installation du Photobooth?

Il faut prévoir un recul suffisant pour la prise de photos, plus pu moins 2 m x 3 m. Il faut egalement prévoir une source
électrique pour l'alimentation des appareils

Est-il possible d'obtenir des impressions instantanées?

Nous offrons l'impression instantanée pour toute location de nos photobooth. Pour plus de renseignements sur les forfaits
vous pouvez-consulter notre site.

Comment puis-je reserver in Photobooth pour mon évenement?

Pour reserver il vous suffit de remplir le formulaire de contact disponible sur notre site ou via les reseaux sociaux.
Nous vous répondrons dans un délai de 3 jours ouvrables.
